Privacy Policy

Effective January 2026

1. Data Sovereignty

Droplet ++ is designed as a "Local-First" tool. Unlike traditional editors that sync your code to a cloud database, Droplet ++ processes your files entirely within your browser environment. Your source code never touches our servers.

2. Cookies and Storage

We use the browser's localStorage API solely to remember your editor preferences, such as theme choice, font size, and active tabs. These stay on your machine and are never shared.

3. External Assets

We use content delivery networks (CDNs) to load libraries like Tailwind CSS and fonts from Google. These providers may receive technical metadata (like your IP address) to serve those assets effectively.

4. Security

Because Droplet ++ uses the File System Access API, your browser will always prompt you for permission before the editor can read or write to a folder on your computer.
